## 开始使用

### 第1步: 登录
```
1. 访问应用首页
2. 输入用户名和密码
3. 点击登录
4. 成功后会获得 Auth Token（自动保存）
```

### 第2步: 进入 SearchParts 页面
```
1. 导航到 "搜索零件" 或 "Search" 菜单项
2. 进入 SearchParts 页面
3. 看到搜索表单和空表格
```

### 第3步: 执行第一次搜索
```
选项 A: 关键词搜索
─────────────────
1. 在搜索框中输入关键词：
   例如: "8407120" (零件号)
       "前罩" (部件名称)
       "A05" (车型)

2. 按 Enter 键或点击 "搜索" 按钮

3. 表格会显示匹配的零件


选项 B: 分类搜索
───────────────
1. 从 "分类" 下拉菜单选择一个分类
2. 点击 "搜索" 按钮
3. 表格会显示该分类下的所有零件


选项 C: 品牌过滤
───────────────
1. 从 "品牌" 下拉菜单选择一个品牌
2. 点击 "搜索" 按钮
3. 表格会显示该品牌的所有零件
```

## 常见操作

### 🔍 搜索操作

#### 关键词搜索
```
场景: 我想找零件号为 "8407120" 的零件
步骤:
1. 在搜索框输入: 8407120
2. 按 Enter 或点击 "搜索"
3. 查看结果

支持的搜索字段:
- 标准名称 (standardName)
- 原始名称 (originalName)
- OE 号 (oeNumber)
- 零件号 (partsNumber)
```

#### 分类搜索
```
场景: 我想看某个分类下的所有零件
步骤:
1. 从 "分类" 下拉菜单选择分类
   例如: "Changan A05"
2. 点击 "搜索"
3. 查看该分类下的所有零件
```

#### 品牌过滤
```
场景: 我只想看长安品牌的零件
步骤:
1. 从 "品牌" 下拉菜单选择 "长安"
2. 点击 "搜索"
3. 查看所有长安品牌的零件

提示: 品牌过滤可以与关键词搜索组合使用
```

### 📊 数据操作

#### 排序
```
最常用的排序场景:

1. 按价格排序 (从低到高)
   □ 点击 "CostInclTax" 列头
   □ 第一次点击 = 升序 (低→高) ↑
   □ 第二次点击 = 降序 (高→低) ↓

2. 按车型排序
   □ 点击 "CarModel" 列头
   □ 快速找到特定车型的零件

3. 按数量排序
   □ 点击 "Quantity" 列头
   □ 找到库存充足或紧张的零件
```

#### 导出
```
导出 CSV 步骤:
1. 完成搜索并排序
2. 点击 "导出CSV" 按钮
3. 浏览器自动下载文件
4. 文件名示例: carparts-1717331200000.csv

导出的文件包含:
- 所有表格列 (9列)
- 当前排序后的顺序
- UTF-8 编码 (Excel 兼容)

用途:
- 报表生成
- 数据分析
- 与其他系统对接
```

#### 打印
```
打印表格步骤:
1. 完成搜索
2. 点击 "打印" 按钮
3. 浏览器打印预览打开
4. 选择打印机或保存为 PDF
5. 点击打印

打印优化:
- 表格会自动分页
- 包含表头在每一页
- 清晰的边界线
- 适应不同纸张大小
```

## ❓ 故障排除

### 问题 1: 搜索无结果
```
可能原因:
□ 搜索词不匹配
□ 后端数据库中没有相关数据
□ API 连接失败

解决方法:
1. 检查搜索词拼写
2. 尝试其他关键词
3. 使用分类搜索代替
4. 查看浏览器控制台错误信息
5. 检查后端 API 是否运行
```

### 问题 2: 表格显示不完整
```
可能原因:
□ 窗口太窄
□ 浏览器缩放
□ 数据字段缺失

解决方法:
1. 全屏显示窗口
2. 重置浏览器缩放 (Ctrl+0)
3. 尝试其他浏览器
4. 查看详情模态框查看完整信息
```

### 问题 3: CSV 导出乱码
```
可能原因:
□ Excel 默认编码设置
□ 系统区域设置

解决方法:
1. 用 Notepad++ 打开 CSV
2. 选择用 UTF-8 BOM 编码
3. 在 Excel 中:
   - 打开 Excel
   - 数据 → 自文本导入
   - 选择 UTF-8 编码
4. 或在 Excel 中手动设置列为 UTF-8
```

### 问题 4: Token 过期
```
症状:
□ API 返回 401 Unauthorized
□ 无法搜索数据

解决方法:
1. 刷新页面
2. 重新登录
3. 新 Token 会自动保存
4. 重试搜索
```
